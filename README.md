<h3>Como rodar o projeto</h3>

```bash
# Clonar repositório
$ git clone https://github.com/KaiqueCovo/todoist-electronjs

# Entrar no repositório
$ cd todoist-electronjs

# Instalar dependências
$ npm install

# Iniciar aplicação
$ npm start

```
